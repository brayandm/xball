## Installation instructions

1 - Copy the .env.example file to .env.local and fill in the variables:

```bash
cp .env.example .env.local
```

2 - Install dependencies:

```bash
npm install
```

3 - Run the application:

```bash
npm run dev
```

## Docker instructions

1 - Copy the .env.example file to .env.local and fill in the variables:

```bash
cp .env.example .env.local
```

2 - Run the application:

```bash
docker compose up
```
